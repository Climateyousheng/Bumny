Adding New Questions

This section summarises the steps to be taken when adding a new question to a GHUI application. See other GHUI documentation for more details of the different types of question allowed, input panel design, filling in variable registers and ways of processing the input. Examples of different types of questions are included in this section.

Each question on an input panel is linked to a particular variable listed in the variable register.
